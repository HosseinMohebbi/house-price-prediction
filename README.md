# House Price Prediction

A supervised machine learning regression project to predict house prices
using the King County housing dataset.

---

## Project Overview
The goal of this project is to build and evaluate regression models that
predict house prices based on various features such as living area,
location, condition, and construction quality.

---

## Dataset
- **Name:** King County House Sales Dataset
- **Target Variable:** `price`
- **Number of Samples:** ~21,000
- **Features:** Numerical and categorical attributes related to houses and their neighborhoods
